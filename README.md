# Lucky Game

---

## What does this project do?

Game-style project.

The user selects a number between 1 and 6. Then, in each round, the program randomly removes one number from the list. If the removed number matches the user's choice, the user loses.

This process continues until only one number remains. If the final remaining number is the one selected by the user, the user wins.

---

## Why did I build it?

I built this project to practice Python fundamentals, especially:
- Working with lists
- Using loops (`while`)
- Implementing conditionals (`if/else`)
- Using randomness with the `random` library

It also helped me understand how to structure a simple game logic.

---

- Interactive user input
- Random number elimination system
- Round-based progression
- Win/lose condition detection
